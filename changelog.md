# Changelog — Federal Marketplace Index
All data updates, method changes, and corrections are recorded here, permanently.

## v0.2 — 2026-07 (exposure table widened to all documented socioeconomic categories)
- `exposure_layers.csv` now reports all nine documented socioeconomic categories —
  small business, SDB, 8(a), WOSB, EDWOSB, HUBZone, SDVOSB, VOSB, and women-owned —
  alongside all federal dollars, up from four (all, small, WOSB, EDWOSB). **No
  previously published figure changed:** the four prior categories reproduce exactly;
  six category columns were added. Same base data (FY2022–FY2026), same award-level
  banding and thresholds, same reconciliation to SBA goaling figures.
- Reading note: for the program categories, the statutory-band (L2) share reflects
  where set-aside–eligible dollars sit. The women-owned column is broader than the
  WOSB program (women-owned firms of any size); read its layers as the share of that
  category's dollars in each band, not as WOSB-program eligibility. Race/ethnicity
  breakdowns are deferred to a forthcoming edition.
- Data as of 2026-07-01, unchanged from v0.1 — this is a coverage/method update, not
  a data refresh.

## v0.1 — 2026-07 (initial public release)
- First publication: protection-layer exposure table and order-channel migration
  series, FY2022–FY2026, from ~30.1M public contract actions reconciled to SBA
  goaling figures. Published to accompany public comments filed in the FAR
  Overhaul rulemakings (FAR Cases 2026-001, -002, -005, -007).
- Known limitations: FY2026 figures are partial-year and labeled as such.
